Sistem Absensi Web

Dikembangkan untuk klinik Wine Dental Aesthetic guna menggantikan absensi manual.

  • Deteksi wajah untuk verifikasi identitas staf saat check-in/out.
  • Pelacakan geolokasi (GPS) agar absensi hanya valid di lokasi kerja.
  • Penanda waktu otomatis pada setiap foto (camera stamp) sebagai bukti kehadiran.
Automasi Skrip Lua

Menulis skrip kustom berbahasa Lua untuk lingkungan game tertentu.

  • Mengotomatisasi tugas berulang agar lebih efisien.
  • Mempelajari logika kondisional dan event-handling di luar konteks akademik formal.
  • Melatih kemampuan debugging dan membaca dokumentasi API pihak ketiga.
Website Biografi Ini

Situs satu halaman ini sendiri adalah proyek untuk Tugas 3 Desain Web, dirancang dengan tema "struk kasir" yang merefleksikan latar belakang hospitality sekaligus minat teknis penulis.

  • Mode terang ("Siang") dan gelap ("Malam") yang bisa dialihkan.
  • Navigasi aktif otomatis mengikuti posisi scroll (scrollspy).
  • Filter proyek, modal detail, validasi formulir, dan mode cetak CV.
